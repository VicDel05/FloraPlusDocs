# Diseño
La interfaz de la aplicación será minimalista, brindando una experiencia agradable e intuitiva para el usuario.

## Home Page
Al iniciar la aplicación se mostrará directamente la vista de la cámara, además de tres botones, los cuales son: cerrar el programa, ver plantas, tomar fotografía.
